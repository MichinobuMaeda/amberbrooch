# Development

## Preriquisites

- Flutter >= 2.5.0
- Node.js >= 14
- [Firebase Local Emulator Suite](https://firebase.google.com/docs/emulator-suite)

```
$ npm -g install yarn firebase-tools eslint jest
$ git clone git@github.com:MichinobuMaeda/amberbrooch.git
$ cd amberbrooch
$ flutter pub get
$ cd functions
$ yarn install
$  cd ..
$ yarn install
$ yarn serve
```
